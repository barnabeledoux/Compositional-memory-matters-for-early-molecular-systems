This repository is made of two folders.

In the folder data/ can be found the data represented in Fig. S13 and S14 and used to produce Fig.4 as a xslx file

In the folder codes/ can be found different python codes to reproduce the figures of the article.

  -class_comparts.py contains the python classes used to simulate the transient compartmentalization process.
  
  -colorsarr.py contains colors used to produce the figures.
  
  -In the folder codes/data/ can be found resluts of the simulation as numpy arrays.
  
  -The folder codes/results/ is used to store figures.
  
  -stirring_exp.ipynb is used to analyze data of the stirring experiments.
